# Cloud Asset Counts

Scripts to count the cloud resources relevant for Uptycs license sizing. Each cloud
provider has its own self-contained script that tallies resource counts and prints or
writes the totals; scripts can be run from the provider's CloudShell or any machine
with the provider's credentials.

Each cloud has its own setup and usage guide:

- **AWS** — [AWS/README.md](AWS/README.md)
- **Azure** — [Azure/README.md](Azure/README.md)

See the per-cloud README for prerequisites, installation, usage, and output details.
